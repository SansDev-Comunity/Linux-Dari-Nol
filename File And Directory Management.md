# Filesystem Hierarchi

/ : menduduki posisi puncak di dalam hiraki.
bin : Direktori ini berisi perintah dasar yang di butuhkan oleh system maupun user.
boot : berisi program dan data yang di butuhkan pada saat melakukan proses booting (menjalankan) system
dev : direktori tempat file device
etc : berisi file konfigurasi system
home : direktori tempat menyimpan data user
var : untuk menyimpan informasi proses, seperti system history, access logs, dan error logs.
mnt : direktori untuk menyimpan mount point sementara.
opt : Direktori ini berisi paket-paket Aplikasi Opsional (paket-paket yang di install user.
