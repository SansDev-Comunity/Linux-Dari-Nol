# Kelebihan dan Kekurangan Linux

1. Pengguna dapat mengembangkannya dengan mudah karena Linux adalah sistem operasi open source
2. Memiliki keamanan yang cukup terjamin, sehingga pengguna tidak perlu khawatir tentang jaminan privasi
3. Selalu melakukan pembaruan secara rutin
4. Proses instalasi yang mudah dan booting yang cepat
5. Ringan untuk dijalankan, sehingga tidak membutuhkan banyak sumber daya dari perangkat komputer
